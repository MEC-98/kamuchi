### Run Locally

To run **KAMUCHI TRANS** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/sabiryosf/kamuchi trans.git
```

Windows:

```bash
git clone https://github.com/sabiryosf/kamuchi trans.git
```
